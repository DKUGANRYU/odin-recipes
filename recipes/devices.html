Devices

Peripherals that connects directly to computer.

Peripherals:
  • Power Strip
  • Monitor
  • Keyboard
  • Mouse
  • Mousepad
  • Headset
  • Speakers
  • Table
  • Computer
Steps:
  1. Get the Table
  2. Put Computer to the Table
  3. Put Power Strip to the Table
  4. Put Monitor to the Table
  5. Put Keyboard to the Table
  6. Put Mousepad to the Table
  7. Put Mouse to the Mousepad
  8. Put Headset to the Table
  9. Put Speakers to the Table
  10. Connect Monitor to Computer
  11. Connect Keyboard to Computer
  12. Connect Mouse to Computer
  13. Connect Headset to Computer
  14. Connect Speakers to Computer
  15. Connect Monitor to Power Strip
  16. Connect Computer to Power Strip
