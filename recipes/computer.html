Computer

Electronic device called Computer.

Computer can run video games.

Components:
  • Case
  • MotherBoard
  • PSU
  • Thermal Paste
  • CPU
  • CPU Cooling
  • GPU
  • RAM1
  • RAM2
  • SSD
Steps:
  1. Open the Case
  2. Put MotherBoard to Case
  3. Put Thermal Paste to MotherBoard
  4. Put CPU over Thermal Paste
  5. Put GPU to MotherBoard
  6. Put RAM1 and RAM2 to MotherBoard
  7. Put SSD to Case
  8. Connect SSD to MotherBoard
  9. Put PSU to Case
  10. Connect MotherBoard to PSU
  11. Close the Case
